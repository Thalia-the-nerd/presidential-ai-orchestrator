Presidential AI Challenge: Setup Your Worker PC

1. Download Client Files

client.py Download requirements.txt Download

Save both files to the root of your portable USB drive.

2. Setup Instructions (The Portable Environment)

A

Install Portable Python (Miniconda)

Install your chosen portable Python distribution (Miniconda/Anaconda) directly onto your USB drive. This step requires only writing privileges to the USB, not admin access on the school PC.

B

Activate the Environment

Open the command prompt, navigate to your portable Python installation, and activate the virtual environment (e.g., .\venv\Scripts\activate).

C

Install Client Libraries

With the environment active, run the following command to install PyTorch and other required libraries:

pip install -r requirements.txt

3. Run and Enroll Your Worker

A

Launch the Client

From the activated environment (and the same directory as client.py), start the worker process:

python client.py
B

Get the Token

The client will print an **8-digit registration token** and wait for approval.

C

Approve the Worker

Log in to the main Orchestrator Management Panel, enter the 8-digit token, and approve the worker to begin submitting tasks!
